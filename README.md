## 🚀 Deployment Instructions

### Deploy to www.plcvijayawada.com

#### Option 1: Netlify (Recommended - Already Connected)
1. Push changes to GitHub
2. Netlify automatically deploys on each push
3. Connect custom domain in Netlify settings

#### Option 2: GitHub Pages
1. Go to repository Settings → Pages
2. Select `main` branch as source
3. Configure custom domain to www.plcvijayawada.com
